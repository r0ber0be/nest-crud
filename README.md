
## Description

A simple Writer crud.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
## Testing the API

- You will need too install the extension REST Client
- Go to the file writers.http
- Run all requests in any order
